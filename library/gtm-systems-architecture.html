Technical Stack: Python (650k+ lines), LLM Orchestration

Impact: Automates arduous parts of market engagement; showcases 'Revenue Architect' capability to ship code-driven strategy.

Project Air Strike

Guerrilla GTM

Core Logic: Monitors technical forums (Reddit, HN, Stack Overflow) for problem-aware signals and users expressing frustration with existing solutions.

Technical Stack: Python, Signal Detection Scripts

Impact: Identifies hidden demand before market saturation; generated 3 SQLs in 48 hours with $0 CAC.

enrich_company.py

Lead Gen / GTM Intelligence

Core Logic: Automates BDR research by taking domain input and performing deep scans for High-Value Signals (tech stack, hiring intent).

Technical Stack: Python, JSON

Impact: Proves 'Data Fabric' capability by structuring raw data for AI readiness; reduces manual research toil; opportunity scores > 80 for automated outbound.

Technical Stack: Narrative Architecture, LLM Prompting

n8n-sentry

Governance

Core Logic: Integrates with Salesforce to enforce 100% CRM integrity, identifies 'Zombie Deals', and logs SLA breaches.

Technical Stack: n8n, Salesforce, Python

Project Sentinel

Partner Channel Activation

Core Logic: Operationalizes dormant partner channels by building signal detection systems and automated partner co-sell motions.

Project Phoenix

Training / Enablement

Core Logic: 5-day 'Live Fire' certification protocol for SDRs utilizing AI-powered training modules and real-time feedback.

Technical Stack: AI Training Modules, Gamified Learning

Impact: Reduced SDR ramp time from 90 days to 5 days (18× improvement); generated 3 SQLs by Day 30.

Outreach Autopsy

Diagnostic / Sales Enablement

Core Logic: Analyzes cold email campaign failures and predicts success rates by scoring messages for engineering credibility and sentiment.

Technical Stack: Python, LLMs (Claude/OpenAI)

Impact: Acts as a 'BS-Detector' to ensure outreach resonates with technical personas and avoids 'sales-speak' rejection; improved reply rates via technical resonance.

Lead-Script-Gen

Lead Gen / Personalization

Core Logic: Generates personalized outbound sequences with specific ICP targeting, automating creative outreach at scale.

Technical Stack: Python, LLMs

Impact: Maintains high relevance in automated outreach while reducing manual time spent on message drafting.

Key Principles

1. Signal Detection First

Every system starts with signal detection—finding problem-aware buyers before they're solution-aware. This is the foundation of all GTM engineering.

2. Automation Over Manual

Replace manual processes with automated systems. From research (enrich_company.py) to outreach (Lead-Script-Gen), automation scales what works.

3. Technical Credibility

In technical sales, showing beats telling. Systems like Outreach Autopsy ensure messages resonate with technical personas, not sales-speak.

4. Data Integrity

Maintain 100% CRM integrity. Systems like n8n-sentry kill "Zombie Deals" and ensure accurate forecasting.

5. Systemization

Turn one-off wins into repeatable systems. Every successful experiment becomes a production system that can be delegated and scaled.
