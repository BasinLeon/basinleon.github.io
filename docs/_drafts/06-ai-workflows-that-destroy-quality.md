Hook: Faster output is not progress if trust drops.

## Problem
- Teams optimize for speed and lose quality control.

## Insight
- Common failure points: weak context, no validation, no review gates.

## Method
- Add schema checks.
- Add human review for outbound messaging.
- Log and review errors weekly.

## Takeaway
Quality debt compounds faster than speed gains.
