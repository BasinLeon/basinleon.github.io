Ahead of Friday's demo, I wanted to send a short note so we have something concrete to react to live.

My current view is that the opportunity here is real, but the core issue is not primarily top-of-funnel lead generation.

You already have enough signal to prove there is demand:
- about 200 users
- strong organic discovery through SEO
- live product and pricing
- early B2B interest from bootcamps, universities, and enterprise

The bigger challenge now is turning that demand into repeat usage, paid behavior, and a B2B offer that institutions will actually buy.

So my working diagnosis is:
- this is a conversion problem
- this is a retention problem
- this is a packaging problem
- this is a GTM operating problem

My recommendation is to split the market for now:
- bootcamps and career accelerators as the near-term primary lane
- universities as a secondary lane for discovery and pilot learning

Here is how I would think about the next 90 days.

30 days
- tighten onboarding and first-session activation
- define the funnel metrics that matter
- split ICP and narrow the B2B wedge
- package a simple pilot offer for institutions

60 days
- test pricing and messaging
- improve repeat usage after the first mock interview
- begin focused outbound and partnership experiments
- sharpen the difference between individual and organization value

90 days
- close the first pilot accounts
- decide which segment converts best
- formalize the repeatable GTM motion
- build around the channels and use cases that actually retain users

On Friday, the main things I want to pressure-test in the demo are:
- the first-time user experience
- what happens after the first interview session
- how users are encouraged to come back
- what an institution would actually be buying beyond individual usage
- where admin visibility, reporting, or cohort management could become part of the value

I’d rather keep this tight and use Friday as a working session.
