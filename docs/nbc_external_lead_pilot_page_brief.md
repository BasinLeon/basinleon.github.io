# NBC External Lead Pilot Page Brief

**Goal:** Create one showable page that Jassi can use in conversations and that can later become a real lead-capture asset.

## Recommended Positioning

This should not be framed as "Leon consulting" or a giant product story.

Frame it as:

- an external lead activation layer
- a low-friction pilot
- an additional source of qualified digital opportunities
- a system that works alongside the current sales motion

## Recommended Page Structure

### 1. Hero

**Headline:** Add a Qualified Lead Layer Without Rebuilding Your Sales System

**Subhead:** External GTM execution for digital sales teams that want more high-fit opportunities routed into the motion they already run.

**Primary CTA:** Request Pilot Walkthrough

### 2. No-Disruption Strip

- No CRM replacement
- No extra SDR buildout required to start
- Pilot-first with baseline comparison

### 3. Three-Step Flow

1. Identify target accounts and buyers
2. Refine and activate against the agreed ICP
3. Hand qualified opportunities to the existing sales team

### 4. Proof Block

Use one short paragraph plus three bullets:

- built for teams shifting from training/readiness into execution
- designed to support existing sellers, not replace them
- measured against speed-to-response and conversion baseline

### 5. Form or Booking CTA

Use a short form:

- Full name
- Work email
- Company
- Role
- Biggest digital pipeline bottleneck

If the page is used in warm conversations only, this can be a booking CTA instead of a form.

## Copy Rules

- One CTA above the fold
- No equal-weight offer grid
- No long founder bio
- No dense methodology section
- No dead-end footer

## Build Sequence

1. Start with a static one-page section or standalone page
2. Add short form capture or a booking link
3. Route responses into the existing handoff flow
4. Add proof or case-study modules only after the core page is live
