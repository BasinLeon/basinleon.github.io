# 1. Open Google Sheets*Created: Monday, August 14, 2023 at 12:43:14?PM*<div><b>1. Open Google Sheets</b></div>
<div>Start by going to Google Sheets (sheets.google.com) and creating a new blank spreadsheet.</div>
<div><b>2. Naming the Spreadsheet</b></div>
<div>You can name the spreadsheet something like &quotJob Search Tracker 2023&quot by clicking on the default name (&quotUntitled spreadsheet&quot) in the top-left corner.</div>
<div><b>3. Define Columns</b></div>
<div>In row 1, enter the following headers:</div>
<ul>
<li>		A1: Company Name</li>
<li>		B1: Position Title</li>
<li>		C1: Date Applied</li>
<li>		D1: Application Status</li>
<li>		E1: Contact Person</li>
<li>		F1: Contact Email/Phone</li>
<li>		G1: Interview Date</li>
<li>		H1: Interview Format (e.g., In-person, Zoom, Phone)</li>
<li>		I1: Offer Status</li>
<li>		J1: Salary Offered</li>
<li>		K1: Notes/Feedback</li>
<li>		L1: Link to Job Posting</li>
<li>		M1: Follow-Up Date</li>
</ul>
<div><b>4. Formatting</b></div>
<ul>
<li><b>Header Row</b>: You might want to bold the header row and give it a background color for distinction.</li>
<li><b>Date Columns</b>: For columns that involve dates (like &quotDate Applied&quot, &quotInterview Date&quot, and &quotFollow-Up Date&quot), you can format them to display dates. Select the column &gt Format &gt Number &gt Date.</li>
<ul>
<li><b>Dropdown Menus</b>: For &quotApplication Status&quot and &quotOffer Status&quot, you can use Data Validation to create dropdown menus:</li>
<li>Click on column D, then go to Data &gt Data validation. For the criteria, choose List of items. Enter Applied,In Review,Rejected,Interview Scheduled,Offered.</li>
<li>Do the same for column I and use Pending,Accepted,Rejected as the list.</li>
</ul>
</ul>
<div><b>5. Conditional Formatting (Optional)</b></div>
<div>You can use conditional formatting to visually distinguish between different statuses. For instance:</div>
<ul>
<li>Highlight &quotApplication Status&quot column, then go to Format &gt Conditional formatting.</li>
<ul>
<li>You can set rules like:</li>
<li>If cell text contains &quotRejected&quot, set a red background.</li>
<li>If cell text contains &quotOffered&quot, set a green background.</li>
</ul>
</ul>
<div><b>6. Freeze Rows and Columns (Optional)</b></div>
<div>It's helpful to freeze the top row so that as you scroll down, you always see your column headers. To do this:</div>
<ul>
<li>Click on the row number 1 to select the entire row.</li>
<li>Then, go to View &gt Freeze &gt 1 row.</li>
</ul>
<div><b>7. Additional Features</b></div>
<div>You can also integrate Google Calendar to remind you of interviews, follow-ups, etc. Additionally, you might want to add a &quotDocuments&quot column where you can paste links to any Google Docs or Google Drive files relevant to that application.</div>
<div><b>8. Regular Updates</b></div>
<div>Remember to regularly update the sheet as you apply for more jobs, hear back from employers, or go through interviews. Over time, this sheet will become a valuable tool in your job search.</div>
