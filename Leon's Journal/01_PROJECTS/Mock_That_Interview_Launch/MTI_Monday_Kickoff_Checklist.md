# MTI Monday Kickoff Checklist

## Goal of the Call
Treat the meeting as a live kickoff and access-handoff, not a generic catch-up.

## Before the Call

- confirm meeting time
- confirm read-only dashboard access
- confirm Google Search Console access
- confirm outreach sheet / CRM access
- gather current landing pages and product URLs
- have the signed agreement and commercial terms handy

## During the Call

### 1. Access handoff
- dashboard
- Search Console
- outreach sheet / CRM
- onboarding assets or demo video

### 2. Baseline capture
- current user count
- weekly signup velocity
- main traffic sources
- first interview completion rate
- known retention or churn issues

### 3. Funnel questions
- where do users stall first
- what defines a successful session
- what makes users return
- what currently drives the best quality signups

### 4. GTM questions
- which institutional lane feels warmest right now
- what outreach has already worked
- which geographies matter most
- where current deals tend to stall

### 5. Sprint alignment
- define first 7-day diagnostic
- define first 1-2 experiments
- define owner for access and follow-ups

## After the Call

- summarize baseline
- identify the biggest bottleneck
- choose first experiment
- choose first outbound list lane
- send follow-up note with next actions

## Recommended First Sprint

1. establish KPI baseline
2. inspect onboarding friction
3. classify first institutional target lane
4. prepare first outbound or competitor-poaching test
