# GTM Email Leadership Playbook

## Core Idea
The job of an important email is to move the motion forward without lowering status.

That means:

- answer the question fast
- frame the real relationship correctly
- make one clean ask
- close without overexplaining

## The Operator Rules

### 1. Answer first
Do not begin with throat clearing.

### 2. Do not apologize by default
If nothing materially broke, skip the apology.

### 3. Name the stage
Kickoff, pilot, partnership, follow-up, negotiation.
Use the correct label.

### 4. Keep asks operational
Calendar invite.
Access.
Sheet.
Payment method.
Baseline data.

### 5. Sound like the person running the machine
Not the person asking to be included in it.

## Use Cases

- client kickoff replies
- founder negotiations
- partner follow-ups
- recruiter replies
- project handoff notes

## Quick Formula

1. answer
2. frame
3. ask
4. close
