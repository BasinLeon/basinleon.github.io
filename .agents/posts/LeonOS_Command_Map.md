# LeonOS: Multi-Agent Command Map

**Governance Rule:** No asset gets shipped because one agent says it is done. It gets shipped when research is sufficient, the idea is clear, the writing is buyer-legible, the artifact is technically ready, and the system of record is updated.

Think of these as default lanes, not permanent handcuffs. Not every task needs every agent.

## The 7-Layer Ecosystem

### 1. Perplexity (The Analyst)
**Lane:** Research
**Job:** Gather current facts, company data, sources, market context.

### 2. Gemini (The Expander)
**Lane:** Expansion
**Job:** Generate angles, models, strategic options, big-picture synthesis.

### 3. Claude (The Refiner)
**Lane:** Prose Refinement
**Job:** Smooth long-form writing, soften rough drafts, improve readability.

### 4. ChatGPT (The Judge)
**Lane:** Judgment
**Job:** Pressure-test clarity, commercial relevance, sequencing, buyer fit, and decision quality.

### 5. Codex / Antigravity (The Executor)
**Lane:** Execution
**Job:** Build files, write code, update repos, scrape, transform, deploy, automate.

### 6. Obsidian (The Vault)
**Lane:** System of Record
**Job:** Store validated frameworks, decisions, outputs, and connected notes for reuse. It is a reference layer, not scripture. Truth is earned through real-world response.

### 7. Grok (The Edge)
**Lane:** Vibe Check
**Job:** Rough pulse on public tone, internet-native framing, social sharpness.

---
**Operating Mantra:** Research with Perplexity. Expand with Gemini. Refine with Claude. Decide with ChatGPT. Ship with Codex. Vibe-check with Grok. Store in Obsidian.
