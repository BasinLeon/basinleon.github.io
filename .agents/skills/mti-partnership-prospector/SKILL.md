# MTI Partnership Prospector

Use this skill when building target lists or outreach lanes for Mock That Interview.

## Objective
Find and prioritize the fastest institutional paths to user growth for MTI.

The default priority order is:

1. bootcamps
2. career accelerators
3. universities

## Output Standard
Every output must classify targets by:

- organization type
- geography
- likely urgency
- likely buyer
- why MTI fits
- next outreach angle

Do not dump generic lists.

## Workflow

### 1. Pick a lane
Choose one segment at a time:

- bootcamps
- career accelerators
- universities

### 2. Find the buyer
Identify the most probable internal owner:

- career services lead
- bootcamp director
- employability manager
- program operations lead
- student success leader

### 3. Score by speed
Prioritize based on:

- urgency around placement outcomes
- visible cohort model
- likely shorter buying cycle
- obvious relevance to interview readiness

### 4. Prepare the angle
Attach one clean reason the target should care now.

Examples:

- improve candidate readiness
- increase placement outcomes
- offer mock interview infrastructure without hiring more coaches
- create a structured AI-assisted prep layer for cohorts

### 5. Hand off cleanly
Return:

- top targets
- rationale
- draft first-touch angle
- any research gaps

## What Good Looks Like

- 10-20 prioritized targets
- clear segmentation
- named probable buyers
- one usable outreach angle per target cluster

## What To Avoid

- giant unscored lists
- founder names with no role logic
- outreach copy without ICP reasoning
- mixing universities and bootcamps into one message

## Principle

The goal is not to find organizations.

The goal is to find the shortest path to an institutional yes.
