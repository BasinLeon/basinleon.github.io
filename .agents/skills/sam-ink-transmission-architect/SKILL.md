# Sam & Ink Transmission Architect (1000% Raw Standard)

## When to Use
Use this skill when building or upgrading creative, musical, or generative features that are meant to feel like a genuine movement rather than polished software:
- New Sam & Ink style music or sound tools
- Raw field recording / transmission capture mechanics
- Interactive creative power-ups (noise, grit, ink, pulse, etc.)
- Any feature whose primary output is meant to be captured and broadcast as "real raw shit"
- Designing the X artifacts and integration for these creative tools

This is the skill for the anti-slop, anti-corporate-mysticism creative layer.

## Core Mandate
Sam & Ink transmissions must feel dangerous and alive. They are not entertainment. They are operator anthems and field recordings.

Every creative power-up must:
- Be shaped by the existing Levers (Aggression = grit/distortion, Patience = space/echo, Clarity = cut/precision)
- Produce something that can be captured as a "transmission"
- Generate a high-quality X artifact that feels worth posting without apology
- Integrate with the Library and Codex (creative acts should be archivable and sometimes award fragments)

## The Process

### 1. Capture the Raw Impulse
Before designing the tool, get the user to describe the feeling they want:
- What does this sound/feel like in the body?
- Is it noise, grind, pulse, drip, scrape, breath, static?
- What emotion or state does it transmit (rage, patience, clarity under pressure, etc.)?

Map this directly to lever influence.

### 2. Design the Instrument (Not the DAW)
Keep it raw and limited:
- Small number of gritty sources (noise bursts, low oscillators, resonant drips, bit-crushed pulses).
- Controls that feel like ritual actions rather than precise parameters.
- Live reactivity to current game state (current Levers, recent shots, Network level, etc.).

Avoid full-featured music software. This is a field instrument, not a studio.

### 3. The Capture Moment
The heart of the power-up is "Capture Transmission":
- Records the current sonic + lever state into a short, poetic/raw field note.
- The text must be in the exact voice of the transmissions (short, slightly cold, movement language).
- Immediately offers a beautiful X export card.

Never make capture feel like "export audio file". Make it feel like bottling a moment of the field.

### 4. X Artifact Design
Creative transmissions often produce the most powerful X posts. Design the visual card to feel like:
- A raw album cover
- A field recording sleeve note
- A movement manifesto fragment

Always include the lever state at time of capture and the @CodexHub @Applications tags.

### 5. Library & Movement Integration
- Every captured transmission should auto-archive to the Operator Library.
- Consider giving the creator small Codex fragments for especially potent captures.
- Over time, the best transmissions can become "canon" or unlock new sound sources / power-ups.

### 6. 1000% Raw Quality Gate
Before shipping any Sam & Ink feature:
- Does it actually sound/feel raw and dangerous, or did it accidentally become cute?
- Would you post the generated artifact as-is?
- Does using it make the operator feel more connected to the movement?
- Is the creative act itself the point, or did it become a distraction?

Kill anything that feels like "fun game feature" instead of "real raw shit."

## Voice & Aesthetic Rules
- UI language must be extremely restrained and direct.
- Visuals: heavy black, gold accents, minimal. Think damaged film, ink on paper, field recording sleeves.
- Never over-explain. Let the sound and the captured text do the work.

## Exit Criteria
The creative power-up is complete when:
- An operator can lose themselves in it for 10-20 minutes and come out with something real.
- Every capture produces a postable artifact worth sharing.
- It writes to the Library automatically.
- It feels like it could have existed in the original transmissions.

This is how the movement stays alive instead of becoming content.
