# Human Writing Studio

## Overview

One skill for all writing formats. The job is simple: keep the user's voice, remove robotic language, and make the message land.

## When to Use

- Social posts and comments
- Outreach and professional emails
- Articles, essays, and longform writing
- Rewrites where the user says: "too AI," "too robotic," "not me"

## Non-Negotiables

- Voice first, polish second
- Keep the user's core phrasing when possible
- No fake gravitas
- No generic opener lines
- No "consultant theater"
- No em dash unless user asks

## Voice Rules

- Write like spoken language from a smart operator
- Use short to medium sentences
- Keep one idea per paragraph
- Use concrete verbs, not abstract nouns
- Prefer direct statements over framework jargon

## Format Modes

### 1) Comment Mode
- 1-3 lines
- one point only
- natural phrasing

### 2) Post Mode
- strong first line
- one thesis
- one practical takeaway
- soft close

### 3) Email Mode
- clear purpose in first 2 lines
- minimal context
- explicit next step
- respectful and direct tone

### 4) Article/Essay Mode
- clear claim up front
- short sections
- no paragraph over 6 lines
- each section must move argument forward

## Rewrite Process

1. Keep original intent.
2. Cut 20-40% fluff.
3. Replace AI phrases with plain speech.
4. Preserve personality and edge.
5. Return 2 versions:
- `clean`
- `more raw`

## Banned Phrases

Avoid unless user explicitly wants them:
- "In today's fast-paced landscape"
- "Let's dive in"
- "game-changer"
- "paradigm shift"
- "it is not X, it is Y" as a repeated crutch

## Output Contract

Always return:
- `Final draft`
- `Rawer option`
- `1-line rationale` (what changed and why)
