# Nexus Retention Moat

The **Nexus Retention Moat** focuses on the 'Leaky Bucket' problem. Growth is useless without retention. This skill engineers the 'Hold Rate'.

## Core Principles

1. **The 90% Hold Rate Target**: Frame all GTM success around user retention, not just user acquisition.
2. **Automated Re-Engagement Drills**: Building logic triggers that fire when a user is inactive for 48 hours.
3. **Diagnostic Loops**: Turning 'empty states' into active CTAs that force the user back into the product workflow.

## Workflow

### 1. Retention Audit
- Identify the exact drop-off point in the user journey (e.g., after the first mock interview).
- Define the 'Retention Critical Path' (RCP).

### 2. Engineering the Moat
- Build 'Diagnostic Values' into the dashboard.
- Create email/push triggers tied to the RCP.

## When to Use
- When a client has 'enough leads' but low repeat usage.
- When shifting from B2C 'Sign-ups' to B2B 'Placement Readiness'.
