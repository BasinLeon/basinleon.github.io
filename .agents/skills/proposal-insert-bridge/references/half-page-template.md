# Half-Page Insert Template

Use this when a partner proposal needs one short section for the operator, specialist, or execution layer.

## Default shape

```md
**[Offer Name] | [Role In The System]**

[80-110 words]
Describe what this layer does after the broader proposal begins. Focus on commercial motion, not biography.

- [Outcome 1 framed as system behavior]
- [Outcome 2 framed as routing / activation / qualification]
- [Outcome 3 framed as benchmark / speed / conversion]

[50-80 words]
Close with the pilot shape. Keep current systems intact, define the small first engagement, and name the comparison logic.
```

## Safe framing patterns

- `execution layer`
- `activation layer`
- `additional source of qualified opportunities`
- `operates alongside the current sales motion`
- `pilot-first`
- `compare against the current baseline`

## Avoid

- long bios
- tool stacks
- partner politics
- "we can do everything"
- immediate CRM replacement
- more than 3 bullets

## Enterprise-safe close

Start with a focused pilot. Tune the ICP, refine the lead or activation logic, route qualified opportunities into the existing motion, and compare the output against the current baseline before expanding scope.
