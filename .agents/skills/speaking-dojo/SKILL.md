# Speaking Dojo

## Overview

Train speaking like a performance skill: short reps, clear feedback, daily consistency.

## When to Use

- Interview performance feels flat or rushed.
- User sounds too technical, too long, or too scripted.
- User wants stronger confidence on live calls.
- User needs concise, high-trust answers.

## Core Principles

- Slow is strong.
- Short beats smart-sounding.
- One message per answer.
- Pause before key lines.

## Daily 20-Minute Protocol

1. `Breath reset` (2 min)
- Inhale 4 sec, exhale 6 sec.
- Drop shoulder tension.

2. `Pace + pause drill` (4 min)
- Read one paragraph out loud.
- Insert a full 1-second pause after each sentence.

3. `Answer compression` (6 min)
- Take one long answer.
- Deliver it in 60 sec.
- Deliver again in 30 sec.
- Keep meaning, remove filler.

4. `Executive framing reps` (6 min)
- Answer with this order:
  - Context (1 line)
  - Action (2 lines)
  - Result (1 line)
  - Why it matters (1 line)

5. `Close strong` (2 min)
- End with one clean line:
  - "That’s the system I’d run here."
  - "That’s how I drive predictable execution."

## Live Call Recovery Rules

If you lose your train of thought:

- "Let me answer that directly."
- "Short version: [one sentence]."
- "The key point is [one sentence]."

If you start rambling:

- Stop.
- Breathe.
- Restart with: "The headline is..."

## Output Format

When this skill is invoked, return:

- `today's drill` (20-minute plan)
- `3 custom prompts` to practice out loud
- `1 concise closing line` matched to role target
