To track your job search like a sales leader, set up a new Airtable Base with the following table and fields:

## Table: "Opportunities"

| Field Name | Type | Purpose |
|------------|------|---------|
| **Job Title** | Single line text | Name of the role |
| **Company** | Single line text | Target company |
| **Status** | Select | [New, Scored, Applied, Interviewing, Offer, Ghosted] |
| **Fit Score** | Number (0-100) | Generated by Gemini |
| **Reasoning** | Long text | AI justification for the score |
| **Cover Letter** | Long text | Generated by Mistral |
| **Talking Points** | Long text | Generated by Groq |
| **URL** | URL | Link to job posting |
| **Date Added** | Date | Automation timestamp |
| **Follow-up Date** | Date | CRM reminder |

## Automation Link
Connect this Airtable Base to your n8n workspace using the **Airtable Node** with your Personal Access Token.
